All Features
A complete overview of every module in the application
🔐
Authentication
Register, login, logout, forgot password, reset password, and email verification. Role-based access for Admin and Pet Owner.
🐕
Pet Management
Add unlimited pets with photos, species, breed, age, weight, vet info, allergies, medical history, and activity level.
⏰
Daily Reminders
10 reminder types: feeding, walking, exercise, grooming, medication, vet appointment, vaccination, training, water, and other. Daily/weekly/monthly repeat.
📊
Dashboard Analytics
Today's progress, weekly activity bar chart, monthly completion donut chart, upcoming appointments and vaccination alerts.
🏥
Health Tracker
Log medical records (checkup, illness, surgery, prescription, weight), vaccinations with due-date alerts, and weight history chart.
📅
Calendar
FullCalendar.js integration showing all reminders and appointments. Switch between month, week, and list views.
🔔
Notifications
Real-time in-app notification bell with unread count. Auto-refreshes every 30 seconds. Mark individual or all as read.
📋
Appointments
Schedule vet visits, grooming, vaccinations, dental, and surgery appointments. Track status: scheduled, completed, cancelled, missed.
👤
Profile Settings
Update name, email, phone, timezone, avatar. Change password. Toggle email notifications, push notifications, and dark mode.
👑
Admin Panel
Platform stats, user management (activate/deactivate), monthly analytics charts, and reports. Accessible only to admin role.
🌙
Dark Mode
Toggle dark/light mode from the top bar. Preference is saved in localStorage and persists across sessions.
🔌
REST API
Full API under /api/v1 with Sanctum token auth. Endpoints for auth, pets, reminders, dashboard, and notifications.
How to Use Each Feature
Step-by-step instructions for every module
🚀
1. Getting Started
Register and set up your account
  • 1
    Open the app
    Go to http://localhost:8000 — you'll see the landing page.
  • 2
    Register or use demo account
    Click "Get started free" to register, or use demo@virtualpetcare.com / password to log in with pre-loaded data.
  • 3
    You land on the Dashboard
    The dashboard shows today's reminders, completion stats, upcoming appointments, and your pets. Everything starts here.
🐕
2. Managing Pets
Add and manage your pet profiles
  • 1
    Go to My Pets
    Click "My Pets" in the sidebar. /pets
  • 2
    Add a new pet
    Click "Add Pet". Fill in name, species, breed, gender, date of birth, weight, and activity level. Upload a photo. Add vet info and medical notes.
  • 3
    View pet profile
    Click "View" on any pet card to see their full profile — age, vaccinations, reminders, vet info, and quick action buttons.
  • 4
    Filter by species
    Use the species badges at the top of the pets list to filter by dog, cat, bird, etc.
💡
The pet age is calculated automatically from the date of birth. Weight is updated automatically when you log a weight record in Health Tracker.
⏰
3. Daily Reminders
Set up and manage care schedules
  • 1
    Go to Reminders
    Click "Reminders" in the sidebar. /reminders
  • 2
    Create a reminder
    Click "New Reminder". Select the pet, choose a type (feeding, medication, walking, etc.), set the time, start date, and repeat schedule (daily/weekly/monthly).
  • 3
    Mark as done
    On the "Today's Schedule" section, click the circle button next to a reminder to mark it complete. It turns green instantly.
  • 4
    Weekly repeat
    When creating a reminder with "Weekly" repeat, select which days of the week it should fire (Mon, Tue, Wed, etc.).
  • 5
    Filter reminders
    Use the dropdowns to filter by type (feeding, medication, etc.), by pet, or by active/inactive status.
💡
Missed reminders are automatically marked at 23:55 each day by the scheduler. You can see your completion rate on the dashboard.
📅
4. Calendar View
See all reminders and appointments in one place
  • 1
    Open Calendar
    Click "Calendar" in the sidebar. /calendar
  • 2
    Switch views
    Use the top-right buttons to switch between Month, Week, and List views.
  • 3
    Click an event
    Click any event on the calendar to see its details in a popup. Purple = reminder, Teal = appointment.
🏥
5. Health Tracker
Log medical records and vaccinations
  • 1
    Go to Health Tracker
    Click "Health Tracker" in the sidebar. /health — you'll see all your pets with their record counts.
  • 2
    Select a pet
    Click on a pet card to open their health page. You'll see the weight history chart (if 2+ weight logs exist), medical records, and vaccinations.
  • 3
    Add a medical record
    Click "+ Add Medical Record" to expand the form. Choose type (checkup, illness, surgery, weight log, etc.), fill in details, and optionally attach a PDF or image.
  • 4
    Add a vaccination
    Click the "Vaccinations" tab, then "+ Add Vaccination Record". Enter the vaccine name, date given, and next due date. The system will alert you when it's due.
  • 5
    Weight chart
    Every time you log a "Weight Log" medical record, the weight history chart updates automatically.
⚠️
Overdue vaccinations show a red "Overdue" badge. Vaccinations due within 30 days show an amber "Due soon" badge on the dashboard.
📋
6. Appointments
Schedule and track vet visits
  • 1
    Go to Appointments
    Click "Appointments" in the sidebar. /appointments
  • 2
    Schedule an appointment
    Click "New Appointment". Select the pet, type (checkup, vaccination, grooming, etc.), date/time, vet name, and clinic details.
  • 3
    Update status
    After the appointment, click "Edit" and change the status to "Completed", "Cancelled", or "Missed".
🔔
7. Notifications
Stay on top of reminders and alerts
  • 1
    Bell icon in top bar
    The 🔔 bell in the top-right shows your unread count. Click it to see the latest 10 notifications. It auto-refreshes every 30 seconds.
  • 2
    Mark as read
    Click any notification in the dropdown to mark it as read. Or click "Mark all read" to clear all at once.
  • 3
    Full notification center
    Click "See all notifications" or "Notifications" in the sidebar to see your full history. /notifications
  • 4
    When are notifications created?
    A notification is created when: you add a reminder, the scheduler fires (every 5 min), or a vaccination is due within 7 days.
💡
Notifications are stored in the database. Email notifications are sent when MAIL_MAILER=smtp and a valid Gmail App Password is set in .env.
👤
8. Profile & Settings
Manage your account preferences
  • 1
    Open Profile
    Click your avatar in the top-right or "Profile" in the sidebar. /profile/edit
  • 2
    Update info
    Change your name, email, phone, timezone, and upload a new avatar photo.
  • 3
    Change password
    Scroll to the "Change Password" section. Enter your current password and new password.
  • 4
    Notification preferences
    Toggle email notifications, push notifications, and dark mode on/off.
👑
9. Admin Panel
Platform management (admin role only)
  • 1
    Log in as admin
    Use admin@virtualpetcare.com / password. The "Admin Panel" link appears in the sidebar.
  • 2
    Admin Dashboard
    See total users, active pets, reminders completed today, and open feedback. Includes weekly registrations and species distribution charts. /admin/dashboard
  • 3
    User Management
    View all users, their roles, pet count, and last login. Activate or deactivate accounts. /admin/users
  • 4
    Reports
    6-month trend charts for new users, new pets, and completed reminders. /admin/reports
REST API Reference
All endpoints are under http://localhost:8000/api/v1. Authenticated routes require Authorization: Bearer {token}
MethodEndpointAuthDescription
POST/auth/registerNoRegister new user, returns token
POST/auth/loginNoLogin, returns Sanctum token
POST/auth/logoutYesRevoke current token
GET/auth/meYesGet current user profile
GET/dashboardYesDashboard stats and data
GET/petsYesList all pets
POST/petsYesCreate a pet
GET/pets/{id}YesGet pet with health records
PUT/pets/{id}YesUpdate pet
DELETE/pets/{id}YesDelete pet (soft delete)
GET/remindersYesList reminders (filterable)
POST/remindersYesCreate reminder
PUT/reminders/{id}YesUpdate reminder
DELETE/reminders/{id}YesDelete reminder
POST/reminders/{id}/completeYesMark reminder as completed
POST/reminders/{id}/snoozeYesSnooze reminder
GET/notificationsYesList all notifications (paginated)
POST/notifications/read-allYesMark all notifications as read
🔑
How to get a token: POST to /api/v1/auth/login with {"email":"demo@virtualpetcare.com","password":"password"}. Use the returned token value in the Authorization header for all subsequent requests.
Tips & Troubleshooting
Common questions and how to fix them
IssueCauseFix
Role does not exist error on registerRoles seeder not runRun php artisan db:seed or php artisan migrate:fresh --seed
Gmail SMTP auth error on registerInvalid App PasswordSet MAIL_MAILER=log in .env for local dev
Notifications bell shows 0No notifications seededRun php artisan db:seed --class=DemoDataSeeder
Notification bell not updatingConfig cache staleRun php artisan config:clear && php artisan cache:clear
Images not showingStorage not linkedRun php artisan storage:link
CSS not loadingAssets not builtRun npm run build
403 on pet/reminder pagesViewing another user's dataEach user can only see their own pets and reminders
Scheduler not runningCron not set upAdd * * * * * php artisan schedule:run to crontab, or run php artisan schedule:work manually
Quick Commands
Useful artisan commands for development
CommandWhat it does
php artisan serveStart the development server on port 8000
php artisan migrate:fresh --seedReset database and seed all demo data
php artisan db:seedSeed roles, admin user, and demo data (without resetting)
php artisan permission:cache-resetClear the Spatie permission cache
php artisan reminders:generate-logsGenerate today's reminder log entries
php artisan reminders:send-notificationsSend notifications for due reminders
php artisan reminders:mark-missedMark past pending reminders as missed
php artisan schedule:workRun the scheduler locally (runs every minute)
php artisan queue:workProcess queued jobs (needed if QUEUE_CONNECTION=database)
php artisan testRun all 22 feature tests
npm run buildBuild CSS and JS assets for production
npm run devStart Vite dev server with hot reload
